Dalam hal ini, a = 1, b = 2, dan c = 2.Karena koefisien a positif (a = 1 > 0), parabola membuka ke atas, yang berarti titik baliknya adalah titik minimum, bukan maksimum. Namun, pertanyaan ini menanyakan titik balik maksimum, sehingga tidak ada titik balik maksimum untuk fungsi ini.Jika kita mencari titik balik minimum, kita dapat menggunakan rumus x = -b/(2a). Dengan nilai a = 1 dan b = 2, maka x = -2/(2*1) = -1.
